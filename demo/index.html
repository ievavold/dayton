It’s just a stylesheet

Consistent web UI doesn’t need to be complicated. We already have the ability to distribute CSS stylesheets to websites and plain CSS can be used with any frontend or backend framework.

The source for this site can be found on GitHub.

Typography

This stylesheet comes with both utility classes and SCSS mixins for configuring typography styles from a fixed set.

Textboxes and dropdowns

Here are some textboxes and dropdowns with some simple styles. No moving labels. No fancy focus ripple.

Plus these basic border, background, and spacing styles are easily composable into more complex input components if applications need those kind of components.
